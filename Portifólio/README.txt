Nesse portifólio eu utilizei Bootstrap 4.
Espero que não seja problema a utilização desse recurso, visto que
me ajudou muito no desenvolvimento do site. Entretanto, eu fiz todo
o HTML, sem copiar nenhum layout ou template já existente.

A seção "Contato" utiliza uma API chamada "PostMail", que disponibiliza
um recurso para envio de e-mail a partir de um "Token". Esse token está
vinculado com meu e-mail, e toda a mensagem enviada chega no e-mail 
cadastrado.
